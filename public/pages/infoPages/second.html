- Loops -

In addition to the traditional for, while, and do-while loops, JavaScript also provides three loop methods that are commonly used when working with arrays:
map, filter, and reduce.

MAP method: The map method allows you to iterate over each element in an array and apply a function to each element.
It returns a new array with the same number of elements where each element is the result of the applied function.

FILTER method: The filter method allows you to iterate over each element in an array and return a new array containing only the elements that meet a specified condition.

REDUCE method: Reduce method allows you to iterate over each element in an array and then reduce it to a single value.
It takes two arguments: a callback function (We talked about that in the "First Server" page), and an optional initial value.

- node_modules -

"node_modules" is a directory that is automatically created by Node.js when you install packages using the npm (Node Package Manager) command.
This directory contains all the dependencies (third-party packages and modules) that your project needs to run.
When you install a package using npm, it gets installed in the node_modules directory of your project.
If the package has any dependencies of its own, those dependencies will also be installed in the node_modules directory, and so on.
This creates a tree-like structure of dependencies, with each package depending on one or more other packages.
